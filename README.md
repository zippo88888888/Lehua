# 提肛运动
> ###### 提肛运动是指有规律地往上提收肛门，然后放松，一提一松就是提肛运动。站、坐、行均可进行，每次做提肛运动50次左右，持续5－10分钟即可。
> ###### 提肛运动可以促进局部血液循环，预防痔疮等肛周疾病（来自百度百科）
